# HoopBiome

HoopBiome is a simple web tool that explains common basketball injuries using basic biology concepts.

Instead of vague injury reports like “ankle sprain” or “ACL tear,” HoopBiome breaks down:

* what actually happens in the body
* recovery timelines
* how performance is affected
* reinjury risk

⸻

🔗 Live Site

https://moe-chi.github.io/hoopbiome/

⸻

💡 Purpose

The goal is to make basketball injuries easier to understand for:

* fans
* players
* anyone curious about performance and recovery

⸻

⚙️ Features (Current Version)

* Injury breakdowns (ACL, ankle sprain, hamstring, etc.)
* Biological explanation of each injury
* Recovery timeline estimates
* Performance impact insights
